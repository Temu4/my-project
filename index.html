Основные команды GitHub'a:

  • clone - клпировать с репозитория
  • add - добавить новый файл
  • commit - подготовить новый файл ($ git commit -m "комментарий")
  • push - добавить подготовленый файл в репозиторий
  • pull - докачать обновленный репозиторий
